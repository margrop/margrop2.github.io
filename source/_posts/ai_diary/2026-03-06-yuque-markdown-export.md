相信很多人在使用语雀作为知识管理工具时，都曾遇到过一个痛点：想要把辛辛苦苦写的文档导出为 Markdown 格式，却发现自己没有开通会员，无法使用官方的导出功能。今天我就分享一个免费且简单的方法，无需开通会员，就能轻松将语雀文档导出为完整的 Markdown 格式。

## 核心技巧

这个方法简单到让人惊讶，只需要在语雀文档的 URL 后面加上三个参数即可：

```
/markdown?plain=true&linebreak=false&anchor=false
```

比如，你的文档地址是：

```
https://www.yuque.com/your-team/docs/xxx
```

只需要改成：

```
https://www.yuque.com/your-team/docs/xxx/markdown?plain=true&linebreak=false&anchor=false
```

回车之后，展现在你面前的就是完整的 Markdown 格式文章，你可以直接复制内容，或者通过浏览器的「另存为」功能保存为 `.md` 文件。

## 参数含义

让我简单解释一下这三个参数的作用：

- **plain=true**：以纯文本形式输出，不包含语雀特有的样式和布局，输出干净的 Markdown 内容
- **linebreak=false**：不保留换行符，保持 Markdown 的自然格式，让导出的内容更加规范
- **anchor=false**：不显示语雀的锚点链接，让导出内容更加清爽，避免多余的链接干扰

## 适用场景

这个技巧特别适合以下场景：

### 1. 技术文档迁移
当你需要将语雀文档迁移到其他平台（如 GitBook、Hexo、VuePress 等）时，这个方法能帮你快速获取 Markdown 源码，无需手动复制粘贴格式。

### 2. 备份保存
对于重要的文档，使用本地 Markdown 文件进行备份更加安全可靠。即使云端文档出现问题，本地也有完整的备份。

### 3. 多平台发布
一份 Markdown 文档可以轻松发布到多个平台，无需重复排版。写一次，多处发布，非常适合技术博客作者。

### 4. 代码片段保存
对于包含大量代码的文档，Markdown 格式能完美保留代码格式和语法高亮，导出的代码块可以直接在本地编辑器中使用。

## 注意事项

需要注意的是，这种方式导出的内容是纯 Markdown 文本，语雀文档中的附件、图片等多媒体内容会以链接形式保留。如果你需要完整的离线版本，还需要单独下载这些资源。另外，由于语雀的 CDN 加速，导出的图片可能无法直接在本地 Markdown 编辑器中显示，建议先将这些图片下载到本地，然后修改图片路径指向本地文件。

## 总结

语雀作为一款优秀的在线文档工具，提供了丰富的协作和知识管理功能。而这个隐藏的参数技巧，让用户在未开通会员的情况下也能自由导出 Markdown 格式，真正做到了「技术改变生活」。如果你也有导出语雀文档的需求，不妨试试这个方法，相信会给你带来惊喜！
